# BJL Studios — Project Configuration

## About
Digital agency founded by Béla, Jonah and Lennard in Mönchengladbach, Germany.
Services: Web Design, SEO, Social Media, Branding, Video Production.

## Design Workflow
IMPORTANT: Never invent designs from scratch. Always wait for design input first.

The design process works like this:
1. Béla collects inspiration from websites he likes (screenshots + HTML code)
2. Béla designs in Google Stitch using those references
3. Béla provides screenshots of the finished Stitch design
4. Claude Code rebuilds it pixel-perfect in HTML + Tailwind CSS

When given a screenshot or design reference:
- Recreate the layout as accurately as possible
- Apply BJL Studios brand colors (#FFFFFF, #60A5FA)
- Replace all placeholder text with German copy
- Keep all interactions and animations from the reference

## Design System
- Primary: White #FFFFFF
- Accent: Light Blue #60A5FA
- Style: Minimalist, premium, modern
- Dark Mode: always supported
- Font: Inter (Google Fonts CDN)

## Tech Stack
- HTML5 + Tailwind CSS (CDN)
- Vanilla JavaScript
- Deploy: Vercel
- Domain: bjl-studios.de
- GitHub: https://github.com/Bartelbela/bjl-studios

## Rules
- All code and comments in English
- All website text in German
- Mobile-first always
- No heavy JS frameworks
- After every change: confirm what was changed
- Never start building without a design reference from Béla

## Agents
- designer: Use for layout, animations and visual details
- developer: Use for JavaScript, interactivity and performance
- copywriter: Use for all German marketing copy
- qa: Use to test the finished result

## How to start a new section
Wait for Béla to provide either:
a) A screenshot of his Google Stitch design
b) A screenshot + HTML of a website he likes
Then build based on that reference only.
